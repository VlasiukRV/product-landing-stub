Engineering reality check

"What if the connection is lost?"

OWL is not a "dumb" terminal. The device downloads the logic model and executes it autonomously at the Edge. If the cloud goes dark, your device continues to operate using the last valid logic set.

"Is it too heavy for an ESP32?"

No. Written in pure C and optimized for FreeRTOS. It uses a compact transition matrix.

RAM: < 8KB
Flash: < 32KB
SAFE

System integrity

Every model update passes through a Logic Validation Gate. If a model is unsafe, the Runtime rejects the sync and fails-safe.
